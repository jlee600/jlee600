## Projects

**[Exoskeleton Control System](https://github.com/jlee600/Exo-Launcher)**  
Distributed control and orchestration for Jetson-based hardware  
- SSH ControlMaster + local API server for low-latency execution  
- Automated multi-device deployment, reducing setup from minutes to ~1 min  
- Real-time telemetry and control across multiple devices  

**[TokenSmith](https://github.com/jlee600/TokenSmith)**  
Local LLM-based retrieval system for textbook-scale PDFs  
- FAISS indexing with hybrid retrieval (dense + BM25)  
- Sub-second semantic search over 100+ page documents  
- Designed for low-latency local inference (llama.cpp / offline pipeline)  

**[CollabPlan-AI](https://github.com/jlee600/CollabPlan-AI)**  
End-to-end speech-to-structure pipeline  
- FastAPI backend with local LLM (Ollama)  
- Audio pipeline: FFmpeg, VAD, Whisper, diarization  
- Generates structured summaries, tasks, and speaker attribution
